The following is a list of networks used by the University of California at Berkeley Campus and some offsite locations. This list is for informative purposes and does not attempt to document all networks in use by Cal Berkeley as some are not managed or provided by Berkeley IT Network Services.

This list can be used to restrict access from specific networks, but be aware that network address are unreliable as a method of data protection.

Full List of Campus Networks

This is the list of networks used by Cal Berkeley and managed by the Berkeley IT Network Services group.
