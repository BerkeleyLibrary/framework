UC Berkeley Campus Networks

    The following is a list of networks used by the University of California at Berkeley Campus and some offsite locations. This list is for informative purposes and does not attempt to document all networks in use by Cal Berkeley as some are not managed or provided by IST Network Services.

    This list can be used to restrict access from specific networks, but be aware that network address are unreliable as a method of data protection.
